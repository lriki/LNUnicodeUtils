LNUnicodeUtils
==============

UTF-8, UTF-16, UTF-32 の相互変換や文字数カウント、不正シーケンスチェックを行うユーティリティクラスです。

Utility class to do the word counting and interconversion, unauthorized check sequence of UTF-8, UTF-16, UTF-32.
